Computer Repair Guides

Find simple step-by-step solutions for common computer problems.

Easy

PC Won't Turn On

Check the power cable, PSU switch, wall outlet, and internal connections.

5 mins

PC Won't Turn On

01

Check the Power Cable

Make sure the power cable is firmly connected to the computer and wall outlet.

02

Check the PSU Switch

Make sure the switch on the back of the power supply is turned on.

03

Test the Wall Outlet

Try another working outlet or test the outlet with another device.

04

Check Internal Connections

Make sure the motherboard and CPU power cables are connected correctly.

Hard

15 mins

Blue Screen (BSOD)

01

Record the Error Code

Write down the stop code shown on the blue screen.

02

Check Recent Changes

Think about recently installed drivers, software, or hardware.

03

Test the RAM

Run a memory diagnostic to check for possible memory problems.

Medium

CPU Overheating

Check cooling, dust buildup, airflow, and your CPU cooler.

10 mins

CPU Overheating

01

Check the Temperature

Monitor the CPU temperature while the computer is running.

02

Clean the Dust

Carefully clean dust from the CPU cooler and case fans.

03

Check the CPU Fan

Make sure the CPU cooler fan is spinning correctly.

Medium

No Display

Check the monitor, display cable, RAM, GPU, and internal connections.

10 mins

No Display

01

Check the Monitor

Make sure the monitor is powered on and set to the correct input.

02

Check the Display Cable

Reconnect the HDMI or DisplayPort cable securely.

03

Reseat the RAM

Turn off the PC and carefully reinstall the memory modules.
